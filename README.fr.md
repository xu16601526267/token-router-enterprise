# token-router enterprise backend

La branche `develop` est une base uniquement backend pour la prochaine reconstruction independante de l'interface. Elle conserve le relais API Go, la gestion multi-locataire, les API keys, le routage fournisseurs/modeles, la comptabilite d'usage, la facturation, le reglement, la gouvernance operationnelle et les taches de fond.

Le code de l'ancienne interface a ete retire de cette branche. Le nouveau travail d'interface doit s'integrer via les API backend.

La description de reference de cette branche est `README.md`.
